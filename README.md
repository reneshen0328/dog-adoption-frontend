# dog-adoption-frontend
A dog adoption online app project for us to practice React and CI/CD deployment. In this app, user can view dogs and subscribe to the adoption center for latest news.
Here's the link to the website: https://reneshen0328.github.io/dog-adoption-frontend

## Overview
<p>This project is the second Multiverse group project, and it's a combination of React practice and CI/CD deployment practice. We created dog-adoption-frontend, JRR Dog Adoption Center, to helps us practice:</p>
<ul>
  <li>Agile planning using Kanban board</li>
  <li>Design UI(user interface) wireframe using Figma</li>
  <li>Design UML(Unified Modeling Language) diagram using whimsical</li>
  <li>Create webpage using React</li>
  <li>Create automated software development workflows using Github Actions</li>
  <li>Monitor code smells using SonarCloud</li>
</ul>
